# Kubernetes Software Versions

Uses semantic versioning

| Major | Minor | Patch |
|------|------|------|
| v1  | .11 | .3  |


- Minor versions are released every few months with new features  
- Patches are released more often  
- Alphas and Betas are released between minor releases
